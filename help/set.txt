proj set — write a key in .projrc

Usage:
  proj set <ProjectName|basename> <key> <value>

Keys are stored lowercase. The value may contain spaces (everything after the key is the value).

Examples:
  proj set MyApp tracker https://github.com/org/repo#issues
  proj set MyApp editor vim          # use named editor from `proj editor`
  proj set MyApp tags web, mobile    # prefer proj tag -p for single-tag changes
